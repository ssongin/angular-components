# Intro

Documentation for developers working on **[Angular Compents](https://github.com/ssongin/angular-components)** project.

## Getting Started

Get started by reading **[user documentation](../documentation/intro.md)** in Documentation section.

Or **try storybooks** for components. Links can be found in same **[Documentation](../documentation/intro.md)** section.

## What you'll need

- Read **[conventional commit messages](./cicd/commit.md)** in Developer section.

- Take a glance at **[tools](./tools.md)** used in project.

- Install [Node.js](https://nodejs.org/en/download/) LTS version.
  - Currently using version 24.0.
